# 📝 To-Do List App

A clean and user-friendly **To-Do List Web Application** built using **Node.js**, **Express**, and **MongoDB**. This app allows users to manage their daily tasks or notes effectively — with features to **add**, **edit**, **delete one**, or **delete multiple** tasks, all stored securely in a MongoDB database.

---

## 🚀 Features

- ✅ Add new tasks or notes with a title and description
- ✏️ Update/edit existing tasks
- ❌ Delete individual tasks
- 🗑️ Select and delete multiple tasks at once
- 💾 All data is saved in MongoDB (persistent storage)
- 🌐 Beautiful, responsive UI with professional design

---

## 🛠 Tech Stack

| Tech         | Usage                            |
|--------------|----------------------------------|
| **Node.js**  | Backend runtime environment       |
| **Express.js** | Web framework for routing        |
| **MongoDB**  | NoSQL database for storing tasks |
| **EJS**      | Templating engine for rendering UI |
| **CSS**      | Styling the frontend              |
| **HTML**     | Markup structure of the app       |

## 🔧 Installation & Run Locally

### Prerequisites
- Node.js & npm installed
- MongoDB installed and running locally

### Steps

```bash
# Clone the repo
git clone https://github.com/your-username/todo-list-node.git

# Navigate to the project directory
cd todo-list-node

# Install dependencies
npm install

# Run the app
node app.js
```
Then open your browser and go to:
👉 http://localhost:3200
